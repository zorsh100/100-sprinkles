# 100 Sprinkles

100 Sprinkles is a vanilla JavaScript K-8 bakery simulator where kids solve adaptive math questions to bake treats, earn coins, and grow their shop. To run it locally, open `index.html` in a browser or serve the folder with any simple static server and load the page there. The game uses a 0-1000 Skill Rating (SR) system to choose age-appropriate question types, keep difficulty near the player's current level, and unlock deeper bakery systems as confidence grows.
